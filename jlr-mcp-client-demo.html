What This MCP Does

The JLR Configurator MCP gives ChatGPT a controlled way to work with the official public configurator payloads behind Range Rover model pages. Instead of open web browsing or generic automotive advice, ChatGPT can ask the MCP for current UK configurator data, summarise builds, compare options, explain dependencies and guide a prospective customer through a buying conversation.

8

MCP tools

Covering guided advice, configurator discovery, build summaries, feature search, dependencies, specifications and comparison.

4

Range Rover model lines

Range Rover, Range Rover Sport, Range Rover Velar and Range Rover Evoque are supported.

UK

Default market

The MCP now defaults to en_gb unless the user explicitly asks for another market.

Why It Matters For JLR

Configurators are rich, but they can be hard for customers to interpret. This MCP lets ChatGPT become a guided product specialist that can reason from JLR's own configurator data whilst staying inside an approved, bounded data source.

From build data to buying confidence

The pilot follows the way a customer naturally thinks: model line, use case, charging access, budget, comfort, performance and equipment. ChatGPT handles the conversation, while the MCP supplies the official configurator facts.

Customer guidance

It can ask about budget, family needs, charging access, driving pattern, towing and priorities before recommending a model.

Official-data grounding

Responses are based on public configurator payloads rather than broad web results, reviews or third-party information.

Decision support

It can compare flagship luxury against PHEV convenience, show trade-offs and provide configurator URLs.

Customer Adviser Flow

The new adviser layer is designed for open-ended questions such as "Which Range Rover should I choose?" If there is not enough information, it asks short buying questions. Once enough context is available, it recommends a UK build and explains the trade-offs.

Understand the brief

Budget, passengers, use case, charging access, towing and priorities.

Resolve UK configurators

Find current public UK model-line URLs and defaults.

Recommend a build

Show model, trim, engine, price, rationale and alternatives.

Refine safely

Preview option changes and dependencies before changing a build.

Pilot Notes

This is a client-demo MCP, not a production customer system. The data boundary is intentionally narrow: public JLR configurator payloads only. A production path would add JLR-approved authentication, governance, monitoring and approved integrations for finance, lead capture, retailer handoff or stock availability.

Recommended next step: run a JLR stakeholder walkthrough using the transcript below, then test three customer journeys: family luxury, company-car PHEV, and towing-focused rural use.

How JLR Can Test It In ChatGPT

The pilot is currently configured for no-auth testing so a small JLR team can try it quickly in ChatGPT. For wider sharing, the same MCP should be moved behind JLR-approved authentication.

Enable Developer Mode

In ChatGPT, open Settings, then Connectors or Apps, and enable Developer Mode from the advanced settings area.

Create a new MCP app

Choose to create an app from a remote MCP server. Use the connector name JLR UK Configurator.

Add the server URL

Use https://jlr-configurator-mcp.onrender.com/mcp and select No authentication for this pilot.

Start a fresh chat

Ask: "Use the JLR UK Configurator MCP. Help me choose the right Range Rover and ask what you need to know first."

Suggested test prompts:
"I have a high budget, a family of four, home charging and I want quiet luxury. Which Range Rover should I choose?"
"Compare full-size Range Rover Autobiography D350 with Range Rover Sport P460e using only the MCP."
"What options should I explore next for comfort, privacy and audio?"

Developer Handover

The working MCP source files have been packaged for review. The bundle contains the Node server, tool adapter, tests, Render deployment files and documentation used for the live pilot.

Developer bundle: jlr-configurator-mcp-dev-bundle.zip
Located in the same folder as this HTML file.
  • src/index.js
    Entry point that selects HTTP or stdio transport.
  • src/httpServer.js
    Remote MCP endpoint, health check, CORS and event-stream compatibility.
  • src/mcpProtocol.js
    MCP initialisation, tool list, tool schemas and tool dispatch.
  • src/rangeRoverConfigurator.js
    JLR configurator adapter, URL resolution, pricing, specs, comparisons and UK adviser logic.
  • src/stdioServer.js
    Local stdio transport for MCP clients that do not use HTTP.
  • test/rangeRoverConfigurator.test.js
    Unit and live smoke tests for markets, specs, prices, dependencies and adviser behaviour.
  • scripts/smoke-http-mcp.mjs
    Simple remote MCP smoke test script.
  • package.json
    Node package metadata and test command.
  • Dockerfile
    Container definition for Render or another host.

    Render Blueprint used for the live no-auth pilot.
  • README.md
